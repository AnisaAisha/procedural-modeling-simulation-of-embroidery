# Procedural Modeling and Simulation of Pakistani Folk Embroidery

**Abstract**: Pakistani folk embroidery represents a rich and regionally diverse cultural heritage, yet its traditional craft techniques are increasingly at risk of being lost to modernization and declining intergenerational transmission. This project proposes a computer graphics approach to digitally preserve and simulate these traditional techniques.The project focuses on developing custom L-system grammars to procedurally generate embroidery textures, render them on physically simulated cloth, and animating the stitching process through parametric curve-based path modeling of needle.
